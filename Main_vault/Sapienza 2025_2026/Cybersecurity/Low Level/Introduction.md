# Information Security
### CIA requirements (The Triad)
- Confidentiality (Optional)
- Integrity [^1] (Always wanted)
- Availability[^2] (Optional)

We don't want all the requirements all the time because more requirements $=>$ More software to write
## Other requirements
- Authentification
- Non-Repudiation [^3]
- Accounting
- Cryptography

[^1]: The data cannot be modified or falsified by an attacker
[^2]: The data is Always available if the user demand it
[^3]: denial of the truth or validity of something
